Project
Afin de retranscrire l'univers de notre photographe, nous avons décidé de montrer les principaux thèmes que celle-ci aimait traiter au lieu d'alourdir le site de centaines de photographies prises de ses mains.

Nous avons construit une maquette puis, après validation par la cliente, avons créé celle-ci sur Angular. Pour dynamiser le tout, des animations ont été implémentées - principalement faites via TypeScript et CSS.

Pour visiter le projet, il vous suffit de cliquer sur le logo d'Esmel Lian, studio de photographie.
